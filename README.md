# Lumora

> **Lumora** é uma landing page integrada a uma loja virtual desenvolvida para apresentar produtos de limpeza de forma moderna, organizada e intuitiva. O projeto conecta um catálogo personalizado ao Mercado Livre, proporcionando uma experiência de compra rápida, segura e focada em praticidade para o dia a dia.

## 📖 Introdução

Encontrar produtos de limpeza de qualidade nem sempre é uma tarefa simples. Muitas vezes, os consumidores precisam navegar por grandes marketplaces até encontrar exatamente o que procuram.

A Lumora foi criada para oferecer uma experiência mais organizada e agradável, reunindo produtos essenciais para limpeza doméstica em uma vitrine moderna, rápida e responsiva. Integrada ao Mercado Livre, a plataforma permite que os clientes descubram produtos com facilidade e finalizem suas compras com toda a segurança da plataforma.

## 🎯 Por que utilizar a Lumora?

* Catálogo organizado de produtos para limpeza doméstica.
* Navegação rápida e intuitiva.
* Design moderno e totalmente responsivo.
* Produtos sincronizados com o Mercado Livre.
* Processo de compra seguro e confiável.
* Experiência otimizada para dispositivos móveis e desktops.

## 💡 Problema que o projeto resolve

Embora marketplaces ofereçam milhares de opções, encontrar produtos específicos pode ser uma experiência cansativa devido ao excesso de informações e anúncios.

A Lumora resolve esse problema ao selecionar e organizar os produtos em uma interface limpa e intuitiva, facilitando a descoberta dos itens e proporcionando uma jornada de compra mais simples e eficiente.

## 🚀 Proposta

O objetivo da Lumora é unir a praticidade de um marketplace consolidado com a experiência de uma landing page de alta conversão. A plataforma apresenta produtos de limpeza de maneira clara, fortalece a identidade da marca e direciona os clientes para uma compra segura através do Mercado Livre.

Mais do que uma simples vitrine online, a Lumora busca oferecer uma experiência de compra agradável, organizada e acessível para quem procura soluções práticas para manter a casa limpa e bem cuidada.
